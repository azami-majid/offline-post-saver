# Offline Post Saver

Offline Post Saver is a Chrome extension for saving individual web posts offline as text or PDF.

It lets you select a post or content block on a page, preview the extracted text, download it as a `.txt` file, or print it as a PDF using Chrome's built-in Save as PDF feature.

## Features

- Select a specific post from any web page
- Save selected post content as plain text
- Save selected post as PDF through Chrome print
- Works on X/Twitter and many other social platforms
- Keeps images and page styling as much as the browser print system allows
- No server, account, or external API required
- Runs locally in the browser

## Use Cases

- Save X/Twitter posts offline
- Archive useful social media posts
- Collect references for research
- Keep important posts before they are deleted
- Export readable copies of web content
